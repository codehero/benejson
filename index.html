benejson by codehero

BeneJSON JSON parsing library.

BeneJSON is a JSON input library with a core written in C and wrapped with a high level C++ interface. The core is meant to be fast, minimal and easy to integrate directly into source. The high level interfaces make it easy to use in C++. In the future, other scripting languages will wrap the core.

Dependencies

libc, STL exceptions

Install

Currently builds with scons. Setup scripts (make, ebuilds, etc) contributions welcome.
